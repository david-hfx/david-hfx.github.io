树莓派初始化配置

发表于 2020-06-03 更新于 2020-06-12 分类于 DIY

过生日女朋友送了一款树莓派4B 4GB版本，闲暇时间就开始折腾起来了，还是挺好玩的，本文的操作环境如下：

  • window10 2004

1. 树莓派开机

我拥有的这款树莓派为无屏款，因此需要搭配电脑和网线使用

1.1. 准备工作

首次开机准备材料：

  • 树莓派
  • 网线
  • 路由器
  • 电脑

    1.2. 开始操作

1.2.1. 烧录系统

准备工作

  • TF卡一张
  • 读卡器一个
  • 下载好的raspberry系统镜像，可选项较多，我选择的树莓派官方系统镜像
  • 烧录软件Win32DiskImager

开干

  1. 格式化TF卡，如果是旧的TF卡，可以用SDFormatter格式化工具先格式化，如果是新的TF卡可以跳过。

    格式化时选择boot分区，格式化就好，格式化完毕如果跳出需要格式化选择取消

  2. 将TF卡插入读卡器插入电脑，打开Win32DiskImager，镜像文件选择下载并解压的系统镜像，设备选择你的TF卡，然后写入，写入后TF卡存在一个boot分区，里面有文件

  3. 在boot分区中添加空的ssh文件

  4. 连接wifi，此步可跳过

    此处参考同济子豪兄

    在boot分区中新建空白文件wpa_supplicant.conf，用文本编辑器打开，写入

1.2.2. 开机

  1. 首先将树莓派安装好，并将网线连接树莓派和路由器的LAN口，形成局域网，电脑连接路由器的wifi，使路由器、电脑及树莓派在同一局域网内

  2. 打开路由器后台，不同的路由器的后台不一致，可以查看路由器背面提示或者上网搜索路由器型号查看，查看当前wifi连接的设备，找到树莓派并记录ip地址

    例如的使用的路由器型号为华为AX3 PRO，电脑打开浏览器，输入192.168.3.1，进入路由器后台查看，可查得树莓派ip地址为192.168.3.5

  1. 下载远程连接工具putty或者xshell，我用的xshell。

    新建一个会话，名称随便填，协议选择SSH，主机填刚才的ip地址，比如我的就是192.168.3.5，端口选择22端口，点击连接

    如果出现红框内字样，则连接成功

1.2.3. 显示图形界面

远程连接成功之后，我们想要开启图形界面，在windows中安装vnc viewer软件，并且在远程登录界面中配置vnc，即可开启，下面讲述具体步骤

  1. 点击链接下载vnc viewer

  2. 在刚才远程登录界面中，执行如下操作

在终端输入

选择Interfacing Options -> VNC -> Yes即可开启VNC服务

  1. 电脑打开vnc viewer，输入树莓派的ip地址，如我的192.168.3.5，输入树莓派的初始用户名和密码，即可成功登录

    用户名：pi

    密码：raspberry

warning：如果出现vnc viewer打开黑屏，可能是因为分辨率设置不对，可以在远程终端中输入sudo raspi-config -> Advanced Optiions -> Resolution -> 合适的分辨率 -> 确定
